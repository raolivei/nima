## Getting Started

### Install Dependencies

```bash
cd frontend
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Usage

1. Enter your prompt in the text area
2. Adjust parameters (Max Length, Temperature, Top K) using the sliders
3. Click "Generate Text" or press `Ctrl+Enter` (or `Cmd+Enter` on Mac)
4. Toggle dark/light mode using the button in the top right

## Keyboard Shortcuts

- `Ctrl+Enter` / `Cmd+Enter`: Generate text

## Color Scheme

- **Purple**: Primary accent color (#8b5cf6)
- **Dark Blue**: Secondary accent (#1e40af)
- **Yellow**: Accent highlight (#fbbf24)

- React 18
